■ 各ツールの主な機能

詳細は各ツールの説明書をご確認下さい。



TMC UV Tool
- TMCファイルのUV座標を変更


TMC Polygon Tool
- TMCファイルのポリゴンメッシュの削除
- TMCファイルのポリゴンメッシュをポリゴンメッシュデータ（tmcmeshファイル）として抽出


TMC Polygon Insert Tool
- TMCファイルへのポリゴンメッシュデータ（tmcmeshファイル）の挿入


TMC Object Tool
- TMCファイル内のオブジェクトの表示非表示フラグを変更
- TMCファイル内のオブジェクトのテクスチャ設定を変更
- TMCファイル内のオブジェクトの透明フラグを変更
- TMCファイル内のオブジェクトの環境マップの有効無効を変更
- TMCファイル内のオブジェクトのMtrCol・matecp・MCAMTRLの割り当てを変更
- TMCファイル内のオブジェクトの描画順を変更
- TMCファイル内のオブジェクトの影を落とす・受けるの有効無効を変更
- TMCファイル内のオブジェクトのUV数を変更
- TMCファイルのオブジェクトグループの追加
- TMCファイルのオブジェクトの追加・削除
- 頂点・Idxグループの追加・削除


TMC Bone Tool
- TMCファイルのボーン位置の移動（Blenderで出力したtmcboneファイルや他TMCファイルを使用）
  ※一部ボーンはモーションで位置が決まるので変更しても変化はありません
- TMCファイルのボーンの親子関係の変更（Blenderで出力したtmcboneファイルや他TMCファイルを使用）
- TMCファイルのコリジョンの変更（他TMCファイルを使用）
- TMCファイルのコリジョンの削除


---C File Editor
- ---Cファイルのオブジェクト表示非表示フラグを変更（TMC Object Toolと併用）
- ---CファイルのAlpha-152用のパラメータを変更


Movie Data Tool
- ボーンとカメラのモーションの簡易編集（ボーンのモーションは編集できないものがあります）
- ボーンとカメラのモーションをCSVファイルとして出力（ボーンのモーションは出力できないものがあります）
- ボーンとカメラのモーションをCSVファイルから入力
- ムービーデータの変更


Texture Tool JPN Custom
- TMCファイル、--Hファイル、--Pファイルなどに設定されているテクスチャを表示・差換え
- TMCファイル、--Hファイルに設定されているテクスチャを追加・削除
- --Hファイルに設定されているテクスチャを空に


TMC Repair Tool
- Toolの過去のバージョンで保存した際のバグでおかしくなったTMCファイルを修復
