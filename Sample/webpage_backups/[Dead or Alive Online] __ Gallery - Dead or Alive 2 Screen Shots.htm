Dead or Alive 2 Screen Shots
:: Kasumi & Tina - New Techniques  

"Kasumi Kyouka" by Nakamura Lucia

         
   

"Michinoku Driver 2" by Takema

         
   


These screens are sequences from the animations for two new holds in DoA2. The cool thing about them is that they were selected by Team Ninja from actual player submissions. Cool!

The first set of images are larger than the rest. The illustrations are scans of the actual player submissions.

Team Ninja seemed pretty enthusiastic about receiving more ideas for new techniques, so I'm reposting the address that was posted to their web site:

ninja@tecmo.co.jp

By the way, that first screen shot has nothing to do with the rest of them. I just needed a convenient place to put it.
:: June 12th Interview Screen Shots  
     
 

This set of screens was posted on June 12th as part of an interview with Itagaki-san. While all of the images obviously feature Helena and a gorgeous new environment, the first in the set provides the first concrete evidence of Zack's existence in Dead or Alive 2.
:: May 8th Screen Shots  
     
   

The following three images were posted to Tecmo's web site on May 8th. Two of them feature the as-of-yet unnamed new character in a new setting with Lei Fang in a new outfit (very new images!). The third is a gorgeous picture of Kasumi with great background detail. The fourth features Kasumi and Ayane, again with great background detail.
:: Environmental Screen Shots  
     

These screens were added for a digital "farewell party" at Tecmo, Japan's web site. They were taken from the same TGS/AOU movie, but they have not, as of yet, been shown as hi-res screens.
:: Tokyo Game Show '99 Screen Shots  
     
     
     
     
 

This set of screen shots was posted to Tecmo, Japan's web site for their 1999 Tokyo Game Show coverage. Of particular interest are the first shots of an as-of-yet unnamed new character.
:: AOU Show '99 Screen Shots  
     
     
 

The following are Tecmo's screen captures from their 1999 AOU Show coverage. The last two images of Kasumi were cut from the movie played at the AOU Show, but were posted for the Tecmo web site.
:: Early Development Screen Shots  
     
     

These screens were the first shots of Dead or Alive 2 posted by Tecmo. For the most part, they are earlier versions of the images taken from the AOU and TGS movies, although a few of the images are not from either existing movie. The last shot depicts what the DoA2 cabinet may look like (and shows that the game is only using three buttons).
